第一章 總則與車輛簡介

1.1 概論

E300 型電聯車為飛灣鐵路之主力通勤車種。本型車設計以安全、舒適及高運量為目標，採用 AC 25kV 架空線供電，最高營運速度為 120 km/h。全車導入數位化 TCMS (列車控制監視系統) 以整合牽引、煞車、空調及 PIS 資訊。

1.2 主要諸元 (Specifications)

供電電壓AC 25,000 V / 60 Hz
編組方式4 輛 (2M2T) / 6 輛 (3M3T)
加減速度加速: 2.5 km/h/s
常用減速: 3.5 km/h/s
緊急減速: 4.5 km/h/s
載客人數 (6輛)定員 1280 人 (座席+立位)
保安裝置ATP / CBTC / TASC / 駕駛失能警惕 (Deadman)

第二章 車輛設備詳解

2.1 駕駛艙介面配置 (Cab Layout)

E300 駕駛艙採人體工學設計，主要操作區分為三大部分：

  • F 區 (前方儀表區)： 包含數位化速度計 (DMI)、ATP 資訊顯示、氣壓表 (MR/BP/BC)、各種指示燈號 (車門、煞車、故障)。
  • L 區 (左側操作台)： 包含 PISC 控制器、無線電手持機、投射燈開關、雨刷控制旋鈕。
  • R 區 (右側操作台)： 包含 TCMS 觸控螢幕、車門開關控制盒、空氣壓縮機強制開關。
  • C 區 (中央主控台)： 主控制器 (Master Controller)、逆轉機鑰匙孔、警惕按鈕、緊急停車按鈕 (E-Stop)。
Page 3

2.2 操控裝置 (Controls & Switches)

2.2.1 主控制器 (Master Controller)

採用左手操作之單把手設計，整合牽引與煞車功能，具備「警惕裝置 (Deadman)」功能。

  • P (Power) 區： P1 ~ P4 四段牽引級位。往後拉為加速。
  • N (Neutral)： 惰行位，無動力輸出。
  • SB (Service Brake) 區： B1 ~ B7 七段常用煞車級位。往前推為減速。
  • EMG (Emergency)： 推至最前方卡槽，觸發緊急煞車，並連動撒沙裝置與電笛長鳴。
注意：手柄設有 Deadman 壓板，行駛中若手鬆開超過 3 秒，將自動觸發緊急煞車。

逆轉機 (Reverser)

位於主控制器右側，需插入司機員鑰匙方可轉動。

  • OFF： 鎖定位置，鑰匙可拔出。
  • FWD： 前進位。
  • NEU： 中立位 (僅供通電但不移動時使用)。
  • REV： 後退位 (限速 25 km/h)。
Page 4

2.2.2 燈光系統 (Lighting System)

燈光開關主要位於駕駛台左側面板 (L 區) 及右側牆面。

設備名稱 位置 操作說明
頭燈 (Headlight) L 區旋鈕 分為三段：OFF / DIM (近光) / HI (遠光)。
正線行駛務必開啟 HI；地下段或會車時切換至 DIM。
標示燈 (Tail Light) L 區撥桿 紅色尾燈。正常運轉時，尾端車自動開啟；
調車或故障處置時可強制手動開啟 (MANUAL ON)。
儀表照明 (Panel Dimmer) F 區旋鈕 可無段調整儀表板背光亮度，夜間或地下段應調暗以防眩光。
客室燈 (Saloon Light) TCMS 螢幕 由 TCMS 控制頁面操作。
模式：全開 (100%) / 減光 (50%) / 關閉。
Page 5

2.2.3 雨刷與視線輔助 (Wipers)

本車配備電動雨刷與清洗液噴射裝置。

  • 控制旋鈕 (Wiper Knob)： 位於左側台。
    • OFF： 停止並自動歸位 (Auto Park)。
    • INT： 間歇刷動 (約 5 秒一次)。
    • LOW： 低速連續。
    • HIGH： 高速連續 (大雨時使用)。
  • 清洗鈕 (Washer)： 按下並保持，噴出清洗液並連動雨刷刷動 3 次。
  • 除霧線 (Demister)： 前擋風玻璃配有電熱除霧功能，開關位於 L 區。

2.2.4 鳴笛裝置 (Horn)

依據用途區分為氣笛 (主要) 與電笛 (次要)。

  • 腳踏開關 (Foot Pedal)： 位於駕駛台下方地板。
    • 輕踩：電笛 (Electric)，聲音較柔和，用於站內或警示月台旅客。
    • 重踩：氣笛 (Air)，聲音宏亮，用於正線警示、平交道或緊急狀況。
  • 面板按鈕 (Panel Button)： 位於 C 區，功能同電笛，供站立或不便踩踏時使用。
Page 6

2.2.5 車門控制面板 (Door Control)

位於駕駛台右側 (R 區) 邊緣，設有防止誤觸之保護蓋。

1. 車門選擇開關 (Door Select Switch)

三段式鑰匙開關：左側 (LEFT) / 切 (OFF) / 右側 (RIGHT)。

  • 必須轉向月台側，開門按鈕才有效用。
  • 行駛中必須置於 OFF 位。

2. 開門按鈕 (OPEN)

黃色自復式按鈕。需在列車速度 < 1 km/h 且 TASC 允許下長按 1 秒方可作動。

3. 關門按鈕 (CLOSE)

黑色按鈕。按下後啟動關門警示音，3 秒後車門開始關閉。

4. 再開門按鈕 (RE-OPEN)

當夾人/物偵測作動或需重新開啟車門時使用，此鈕無 TASC 鎖定限制。

5. 側燈 (Side Indicators)

車外橘色指示燈，亮起代表該車門未完全鎖定。

2.3 PISC 旅客資訊系統 (PISC)

(Passenger Information System Controller)

本系統負責整合 LED 顯示器 (車頭/車側/車內)、自動播音 (DVA) 及車廂對講機。舊版操作介面已更新，請依照下列標準程序操作。

2.3.1 系統架構

  • PCU (主控制單元)： 位於 Tc1 車，負責儲存路線資料庫。
  • HMI 觸控螢幕： 司機員操作介面。
  • SND (備用實體鍵盤)： 若觸控螢幕故障時之備援操作盤。

2.3.2 操作介面說明

功能頁籤功能描述
運行設定 (RUN)輸入車次、起訖站、司機員代碼。
狀態監控 (STATUS)顯示各車廂廣播音量、LED 顯示狀態是否正常。
緊急對講 (PEI)接聽旅客緊急通話 (紅燈閃爍時點擊接聽)。
維修模式 (MAINT)僅供維修人員更新資料庫或測試單一喇叭使用。
Page 8

2.3.3 標準設定流程 (Setup Procedure)

請於始業檢查通電後依序執行：

Step 1: 系統登入

開機後點選 [登入]，輸入司機員 ID (例如: 88001) 並按 [Enter]。

Step 2: 載入班次

進入 [運行設定] 頁面，輸入 車次號 (Train ID) (例如: 101)。系統將自動帶出起點與終點。

Step 3: 確認路線 (Route Select)

確認螢幕顯示之停靠站列表正確。若需變更 (如跳站、臨時改道)，請點選 [手動選擇] 修改。

Step 4: 啟動模式 (Mode)

確認模式設定為 [AUTO]。此模式下，系統將依據 GPS 與里程計訊號自動觸發離站/進站廣播。

2.3.4 特殊狀況操作

  • 手動觸發 (Manual Trigger)： 若 GPS 失效，請切換至 [MANUAL]，進站前按壓實體 [綠色按鈕 (ANN)] 播放進站廣播。
  • 臨時停車 (Delay)： 遇號誌臨停超過 1 分鐘，按壓實體 [黃色按鈕 (TEMP)] 播放道歉廣播。
  • 更正站名： 若系統顯示站名與實際不符，使用螢幕上之 [上一站] / [下一站] 箭頭進行修正。

2.4 保安與信號裝置 (Safety Systems)

E300 配備多重冗餘之安全防護系統，確保列車運行絕對安全。

2.4.1 ATP (Automatic Train Protection) 列車自動防護系統

ATP 為本車最核心之保安裝置，其功能為連續監控列車速度，防止超速與闖紅燈。

ATP 作動原理： ATP 電腦根據前方閉塞區間狀態、路線速限及坡度，計算出一條「動態速度曲線 (Dynamic Profile)」。
  • 監控模式：
    • MCS (Master Control): 正常模式，依據速限行駛。
    • RM (Restricted Manual): 限制手動模式，速限 25 km/h (無信號時使用)。
  • 超速防護：
    • 當實際速度 > 允許速度 + 2 km/h：警示音 (Alarm) 響起。
    • 當實際速度 > 允許速度 + 5 km/h：常用煞車 (SB) 自動介入直到速度低於限速。
  • 緊急停止： 若列車越過紅燈 (EOA, End of Authority)，ATP 將觸發緊急煞車 (EB) 直至列車完全停止。
Page 10

2.4.2 CBTC (通訊式列車控制)

透過無線電 (WLAN) 即時與地面設備通訊，實現移動閉塞 (Moving Block)。

  • MA (移動授權)： 駕駛台儀表上的「MA Bar (綠色條)」顯示前方可通行距離。若 MA 歸零，列車不得移動。
  • ATO (自動列車駕駛)： (選配功能) 若啟用 ATO，司機員僅需監控，列車將自動加速與減速。

2.4.3 TASC (定點停車輔助)

協助司機員將列車精準停靠於月台門位置。

  • 作動時機： 進站前約 300 公尺偵測到 TASC 地面感應器。
  • 操作： 司機員需將電門保持在 P 位或 N 位，TASC 燈亮起後，系統自動接管煞車將列車停準。
  • 停準確認： 停車後，儀表板顯示「TASC OK」綠燈，且誤差在 ±30cm 內，車門方可開啟。

第三章 標準操作程序 (SOP)

3.1 始業檢查 (Pre-Departure Check)

本程序於每日出庫前執行，確保列車各系統功能正常。

A. 駕駛室靜態檢查

1. 電瓶電壓 (Battery Volt)確認 > 90V (低壓表)
2. NFB 配電盤確認全數投入 (ON)
3. 手軔車 (Parking Brake)確認施軔 (紅燈亮)
4. 駕駛台開關位置全數歸零 / OFF 位
5. 滅火器/擊破器確認在位且壓力正常

B. 設備功能測試 (低壓)

1. 司機員鑰匙插入 -> ON
2. 燈光測試頭燈(遠/近)、尾燈、儀表燈 正常
3. 雨刷測試噴水、間歇、高低速作動正常
4. 鳴笛測試電笛(輕踩)、氣笛(重踩) 響聲正常
5. 廣播測試執行 PISC 廣播測試，車內外喇叭正常
Page 13

C. 車外巡視檢查 (Exterior Walk-around)

司機員須下車繞行列車一週進行目視檢查。

1. 車體外觀無外傷、塗鴉、外板脫落
2. 轉向架/避震器無漏油、無異物卡滯
3. 連結器 (Coupler)電氣接頭清潔、氣管無洩漏
4. 車下設備箱箱門鎖緊、無燒焦異味
5. 集電弓 (Pan)碳刷磨耗正常、無纏繞異物
特別注意： 檢查集電弓時，務必確認集電弓處於降弓狀態，且人員與帶電部分保持 1.5 公尺以上安全距離。
Page 14

3.2 系統啟動與 PISC 設定

A. 高壓送電 (Energizing)

1. 集電弓開關 (Pan)操作 [UP]，確認升弓燈亮
2. 線電壓 (Line Volts)確認檢出 25kV (儀表顯示)
3. VCB 開關操作 [CLOSE]，聽見閉合聲
4. 輔助電源 (SIV)確認 SIV 指示燈亮 (綠燈)
5. 空壓機 (CMP)自動啟動充氣，至 8.5 bar 停止

B. 資訊系統設定

  1. 於 PISC 觸控螢幕登入司機員 ID。
  2. 輸入本日車次號 (Train ID)。
  3. 確認起訖站與停靠模式 (每站停/直達)。
  4. 設定空調溫度 (標準 24°C)。
  5. 確認 TCMS 顯示全車設備「綠色 (正常)」。
Page 15

3.3 軔機與保安裝置測試

(Brake & ATP Function Test)

A. 軔機貫通試驗

1. 電門 -> SERVICEBC 壓力上升，BP 壓力下降
2. 電門 -> EMGBC 最大壓 (4.5bar)，BP 排至 0
3. 電門 -> RELEASEBC 歸零，BP 回充至 5.0bar
4. 手軔車 (Parking)操作 緩解/施軔 各一次，確認燈號

B. ATP 功能自趨 (Self-Test)

開啟 ATP 電源後，系統會自動執行自我診斷。

  • 觀察 DMI 螢幕，確認無故障代碼 (Error Code)。
  • 確認 ATP 模式顯示為 STBY (待機) 或 MCS。
  • 測試「警惕裝置」：按下警惕鈕，確認警報音消失。
Page 16

3.4 出庫與正線行駛

A. 發車前確認 (Departure Check)

1. 車門狀態全關 (All Closed) 燈亮
2. 氣壓狀態MR > 7.0 bar / BP 5.0 bar
3. 逆轉機FWD (前進位)
4. 手軔車緩解 (RELEASE) 燈滅
5. 移動授權 (MA)確認 DMI 顯示可行駛距離
6. 指差確認呼喚「出發號誌，綠燈/進行！」

B. 行駛操作要領

  • 起步： 將電門推至 P2 或 P3，列車移動後再視情況增加至 P4。
  • 定速： 達到目標速度後，將電門拉回 N (惰行) 或維持低檔位。
  • 減速： 進站前預先將電門推至 B1~B3 建立煞車力，避免急煞。
  • 停站： 配合 TASC 系統，將車輛準確停於停止線 ±30cm 處。
Page 17

3.5 洗車與調車作業

A. 洗車程序 (Washing Mode)

進入自動洗車機前之操作：

  1. 確認所有車門、車窗緊閉。
  2. 關閉雨刷 (置於 OFF 位)。
  3. 於 TCMS 空調頁面點選 [洗車模式] (此動作將關閉外氣風門，防止藥水吸入)。
  4. 將 ATP 切換至 RM 模式。
  5. 維持 3~5 km/h 恆速通過洗車機。

B. 調車程序 (Shunting)

於機務段內進行車輛移動：

  • 速限： 機廠內速限 25 km/h；進入維修坑道速限 5 km/h。
  • 喚呼應答： 確實覆誦調車員之無線電指令。
  • 燈光： 開啟雙邊尾燈 (若有規定) 或頭燈近光。

3.6 收班與斷電程序 (Shutdown)

當日運轉結束，列車返回停放股道後執行。

1. 停車定位確認停於規定位置 (止衝檔前)
2. 手軔車施軔 (SET) 紅燈亮
3. PISC 關機執行登出並關閉系統
4. 燈光/空調全數關閉
5. VCB 切斷操作 [OPEN]
6. 降弓操作 [DOWN] 確認降弓燈亮
7. 司機員鑰匙轉至 OFF 並拔取
8. 電瓶開關 (Bat)切斷 (OFF)
9. 排水 (Drain)執行 MR 儲氣槽排水 (車外)

第四章 簡易故障排除

4.1 故障處置原則

遇故障時，請遵循「RESET (重置)」原則：

  1. 確認故障訊息代碼 (TCMS Event Code)。
  2. 參閱故障排除手冊 (Troubleshooting Guide)。
  3. 若無立即危險，嘗試重置相關 NFB 或 VCB。
